# Welcome to my extention!
Go yo your extention page then load the unpacked extention folder

1. Load the Extension in Chrome
Open Google Chrome.
Navigate to chrome://extensions/.
Enable Developer mode (toggle in the top right).
Click Load unpacked.
Select the folder containing the manifest.json

You should now see your extension appear in the list and its icon in the toolbar!

3. Using the Extension
Visit the Stoneworks Map once you have enabled the extention

4. Updating the Map
To update the map go to the stoneworks map
Go into the devmode
Go to network
Download the markers.json the webmap is using(turn off the extention to confirm your getting the right one)
